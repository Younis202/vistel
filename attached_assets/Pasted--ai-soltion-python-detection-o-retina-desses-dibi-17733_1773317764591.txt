You are a senior AI engineer, machine learning architect, and medical imaging specialist.

Your task is to design and implement a full AI ophthalmology diagnosis system similar in concept to systems built by companies like Vistel.

The goal is to analyze retinal fundus images and generate a medical diagnostic report using a multi-model AI pipeline.

The system must be modular, production-ready, and scalable.

PROJECT GOAL

Build an AI system that:

Accepts retinal fundus images from users.

Runs multiple AI models to analyze the image.

Detects retinal diseases.

Produces a structured medical report.

DATASETS

Use publicly available retinal datasets for training:

EyePACS

Messidor

ODIR

APTOS 2019 Blindness Detection

Dataset structure:

dataset/
images/
labels.csv

labels.csv format:

image_id,diagnosis

Example:

img_001,normal
img_002,dr_mild
img_003,dr_severe

IMAGE PREPROCESSING PIPELINE

Implement preprocessing steps:

Remove black borders

Retina circular cropping

Resize images to 512x512

Normalize pixel values

Apply CLAHE contrast enhancement

Image augmentation:

horizontal flip

rotation

brightness adjustment

zoom

DATA SPLIT

Split dataset into:

70% training
15% validation
15% testing

Avoid data leakage.

AI MODEL PIPELINE

The AI system must consist of multiple models.

MODEL 1 — Image Quality Assessment

Goal:
Detect if retinal image quality is usable.

Architecture suggestions:

EfficientNet-B0

Output:

good_quality / bad_quality

MODEL 2 — Retina Segmentation

Goal:
Segment key anatomical structures.

Segment:

optic disc

macula

retinal vessels

Architecture:

U-Net or UNet++

Output:

segmentation mask

MODEL 3 — Lesion Detection

Goal:
Detect pathological features:

microaneurysms

hemorrhages

exudates

Architecture options:

YOLOv8
or Faster R-CNN

Output:

bounding boxes of lesions

MODEL 4 — Disease Classification

Goal:
Predict retinal diseases.

Target diseases:

Diabetic Retinopathy

Age-Related Macular Degeneration

Glaucoma

Normal retina

Architecture:

ResNet50 or DenseNet121

Output:

probability scores for each disease.

MODEL ENSEMBLE

Combine model outputs into a final risk assessment.

Pipeline:

Retina Image
→ Quality Model
→ Segmentation Model
→ Lesion Detection
→ Disease Classifier
→ Risk Score Engine

EXPLAINABILITY

Add explainable AI features:

Grad-CAM heatmaps

Saliency maps

Doctors must see which regions influenced the prediction.

AI OUTPUT

The AI should produce:

disease prediction

severity score

highlighted lesion areas

confidence percentage

BACKEND ARCHITECTURE

Use Python and PyTorch.

Framework suggestions:

FastAPI for AI server.

API endpoints:

POST /analyze-image
GET /report/{id}

MODEL STORAGE

Save trained models as:

model.pt

Load them dynamically in the inference pipeline.

WEB APPLICATION

Build a simple medical web interface.

Features:

Upload retinal image

Run AI analysis

Display:

diagnosis

heatmap visualization

detected lesions

REPORT GENERATION

Generate a PDF medical report including:

Patient ID
Retina image
AI diagnosis
Detected lesions
Confidence score
Timestamp

INFRASTRUCTURE

Training:

GPU support (CUDA)

Suggested GPUs:

NVIDIA A100
NVIDIA L4

Use cloud training if needed.

PROJECT STRUCTURE

project/

data/
models/
training/
inference/
api/
web/
reports/

FINAL OUTPUT

Deliver a working AI system capable of:

retinal image analysis

multi-model medical inference

automated report generation

Ensure code is modular, documented, and production-ready.
